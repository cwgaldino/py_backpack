interact

Functions for user interaction.

backpack.interact.operating_system()

Return string with name of operating system (windows, linux, or mac).

backpack.interact.make_sound(duration=1, freq=440)

Make a sound.

On linux (tested on ubuntu) it uses sox (sudo apt install sox). On windows it uses the python package winsound (pip install winsound).

Parameters

  • duration (int, optional) – duration in seconds.

  • freq (int, optional) – frequence of sound in Hertz

backpack.interact.say(message)

Text-to-speech function.

On Linux speech-dispatcher package is necessary (sudo apt-get install -y speech-dispatcher).

On windows it uses wsay package (for more information see https://github.com/p-groarke/wsay).

Parameters

message (str) – message to be said out loud.

backpack.interact.query(question, default='yes')

Ask a yes/no question and return answer.

Note

It accepts many variations of yes and no as answer, like, “y”, “YES”, “N”, …

Parameters

  • question (str) – string that is presented to the user.

  • default ('yes', 'no' or None) – default answer if the user just hits <Enter>. If None, an answer is required of the user.

Returns

True for “yes” or False for “no”.

backpack.interact.copy2clipboard(txt)

Copy text to clipboard.

on linux it uses xsel package (sudo apt-get install -y xsel).

backpack.interact.png2clipboard(filepath)

Copy png pictures to clipboard.

Warning

Only implemented on Linux.

On linux it uses xsel package (sudo apt-get install -y xsel).

backpack.interact.svg2clipboard(filepath)

Copy svg pictures to clipboard.

Warning

Only implemented on Linux.

On linux it uses xsel package (sudo apt-get install -y xsel).

backpack.interact.send_email(recipient, subject, body, sender, password=None)

Send email.

If using a gmail sender email turn Allow less secure apps to ON. Be aware that this makes it easier for others to gain access to your account.

Warning

Be aware that writting a password on a python script is a bad practice. It is recomended creating a development email just for usage with this function and never share sensitive information via this account.

Parameters

  • recipient (str) – email of the recipient.

  • subject (str) – email subject.

  • body (str) – email body.

  • sender (str) – sender email.

  • password (str) – password of the sender email. If None, it will securely ask the user to input the password.
