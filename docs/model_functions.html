Model functions

Mathematical functions and distributions.

backpack.model_functions.gaussian(x, amp, c, sigma)

Gaussian distribution.

\[y(x) = \text{amp } e^{-\frac{(x-c)^2}{2 \sigma^2}}\]

where,

\[\text{Area }= \sqrt{2 \pi} \text{ amp } |\sigma|\]

and,

\[\text{fwhm }= 2 \sqrt{2 \ln(2)} \sigma\]
Parameters

  • x – x array

  • amp – Amplitude

  • c – Center

  • sigma – standard deviation

Returns

\(y(x)\)

backpack.model_functions.lorentzian(x, gamma, c)

Cauchy–Lorentz distribution.

\[y(x) = \frac{1}{\pi \gamma} \frac{\gamma^2}{\gamma^2 + (x-c)^2}\]

where,

\[\text{Amplitude }= \frac{1}{\pi \gamma}\]

and,

\[\text{fwhm }= 2 \gamma\]

and,

\[\text{Area }= 1\]
Parameters

  • x – x array

  • gamma – Scale factor

  • c – Center

Returns

\(y(x)\)

backpack.model_functions.voigt_area_fwhm(x, A, c, w, m)

Pseudo-voigt curve.

\[y(x) = A \left[ m \frac{1}{\pi w} \frac{w^2}{w^2 + 4 (x-c)^2} + (1-m) \frac{2 \sqrt{\ln(2)}}{w \sqrt{\pi}} e^{-\frac{4 \ln(2) (x-c)^2}{w^2}} \right]\]

where,

\[\text{amp }= \frac{A}{w} \left[ m\frac{1}{\pi} + (1-m)\frac{2\sqrt{\ln(2)}}{\sqrt{\pi}} \right]\]
Parameters

  • x – x array

  • A – is the Area

  • c – Center

  • w – FWHM

  • m – Factor from 1 to 0 of the lorentzian amount

Returns
